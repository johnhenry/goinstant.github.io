Integrations

Open source libraries we’ve created based on GoInstant’s platform. Check out our documentation or the individual repos for more info.

Ouija

An inline commenting system built on top of Ghost. Ouija is built using React.

GoAngular

AngularJS + GoInstant. Focus on building your Angular application, we've got your back(end).

GoRTC

The GoRTC library glues together WebRTC and GoInstant to offer video and audio conferencing with no server deployment and configuration. This makes it incredibly easy to get started with WebRTC in your application.

Widgets

GoInstant widgets are a set of easy-to-use, drop-in feature enhancements for any application. They bring multi-user, realtime experiences to life!

Chat

Real-time messaging inside a room of your application.

User List

List of users connected inside a room of your application in real-time.

Rich Text Editor

A real-time rich text editor (RTE) for collaborative text editing inside a room of your application.

Audio and Video

Provides a drop-in, back-end free video conferencing solution for your app. It's powered by GoRTC library, our integration with WebRTC.

Collaboration Bundle

Combines GoInstant widgets together and adds new functionality to make instant collaboration on your website possible.

Form

Integrates real-time, multi-user capabilities into a form, allowing two or more people to fill it out together.

Notifications

Real-time growl-like messages inside your application.

Click Indictator

Provides real-time click animations for your application.

Scroll Indicator

Shows a notification when remote users that you're collaborating with scroll their browser viewport, and when they're viewing a different part of the page.

User Colors

The User Colors library allows you to assign a unique color to each user in a room
